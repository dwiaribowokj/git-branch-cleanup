# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Expected behavior

- Be kind and constructive.
- Assume good intent.
- Keep feedback specific and actionable.
- Respect different experience levels.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Discrimination or exclusionary behavior.
- Publishing private information or secrets.
- Spam or intentionally disruptive behavior.

Maintainers may remove comments, close issues, or block users when needed to keep the project healthy.
